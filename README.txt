1) Eclipse Project (ImageTouchDrive_00001.rar) with an Android 
application that shows how to customize a view and allow the 
user to touch drive bitmaps on the screen. 

2) The application draws 3 black and 3 white chess pieces.
The user should draw the three black pieces down in such
a way that no white piece jumps on the black piece that
is being driven down.

3) Targets: Android 2.3+, Eclipse Helios 3.6

4) To start: 
   a) unzip into a folder 
   b) open in Eclipse 
   c) Run as Android application

5) bitmap images of chess pieces are taken from 
http://en.wikipedia.org/wiki/Chess_symbols_in_Unicode
